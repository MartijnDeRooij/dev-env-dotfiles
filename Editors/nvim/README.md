# 📎 Neovim Setup

> **Important**
>
> This repository uses **lazy.nvim**, **Mason**, and **nvim-treesitter**.
>
> When starting Neovim for the first time, allow plugins, language servers, and parsers to install automatically. If you open a file before Mason finishes installing its language servers, you may see startup errors. Press **Enter** and wait for Mason to complete the installation. If you clone the repo into your machine and use the config by copying .config/nvim to your home folder.
